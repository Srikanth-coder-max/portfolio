📊 ML · Anomaly Detection

Real-Time Infrastructure Health Analytics & Anomaly Detection

GitHub

▸ PROBLEM

Manual monitoring of production infrastructure led to undetected CPU/memory spikes, causing unplanned system outages with no early warning.

▸ APPROACH

Built a psutil OS-level pipeline collecting live telemetry → cleaned into normalised SQLite schema → trained a Random Forest model on labeled anomaly windows with cross-validated evaluation.

▸ RESULT

30–50% reduction in missed anomaly spikes, measured over a 2-week synthetic load simulation with labeled anomaly injection. 20–40% improvement in resource utilisation via proactive alerting. Live Streamlit dashboard deployed for ops teams.

Engineering struggle: Faced noisy telemetry causing false positives; reduced alert noise using rolling feature smoothing and calibrated threshold tuning.

Python Scikit-learn Random Forest Pandas / NumPy SQLite Streamlit psutil
🎙️ RAG · NLP · Agents

ContextualVoice — RAG Voice AI with Persistent Memory

GitHub

▸ PROBLEM

Standard LLM chatbots hallucinate and lose context across multi-turn conversations — unusable for persistent task orchestration or domain-specific Q&A.

▸ APPROACH

Engineered a modular Brain layer routing inputs through a ChromaDB RAG pipeline (semantic search + short-term history fusion) with voice I/O via Faster-Whisper STT and pyttsx3 TTS.

▸ RESULT

Achieved grounded, context-aware responses for domain Q&A by anchoring all LLM outputs in retrieved memory. Multi-tool agent supports web search, weather, and system monitoring in real time.

Engineering struggle: Retrieval quality dropped on short, ambiguous voice prompts; fixed with query rewriting plus hybrid reranking before final generation.

Python LangChain ChromaDB (Vector DB) RAG Faster-Whisper LLMs (Groq/Ollama) REST APIs
